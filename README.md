# Telaraña en Batalla

Un minijuego arcade en HTML5 donde controlas a una araña guardiana. Come la mosca para sumar puntos,
lanza seda para ahuyentar a las arañas intrusas y evita que te golpeen.

## Cómo jugar

1. Abre `index.html` en tu navegador favorito.
2. Mueve a la araña con **WASD** o las **flechas**.
3. Apunta con el ratón y pulsa **Espacio** o haz clic para disparar.
4. Come las moscas para curarte un poco y ganar puntos.
5. No dejes que las arañas enemigas alcancen a la mosca ni que te golpeen.

Pulsa el botón **Reiniciar** si pierdes toda la resistencia.
